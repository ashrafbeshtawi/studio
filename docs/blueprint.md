# **App Name**: Berlin Bytes

## Core Features:

- Intro Section: Display personal introduction, including name and a brief summary.
- Skills Showcase: Showcase skills, tech stack, and languages.
- Projects & Contact: Display links to projects and contact information.

## Style Guidelines:

- Primary color: Dark gray (#333) for a professional look.
- Secondary color: Light gray (#f0f0f0) for backgrounds and subtle accents.
- Accent: Teal (#008080) to highlight key elements.
- Clean and modern layout with clear sections for easy navigation.
- Subtle transitions and hover effects to add interactivity without being distracting.
- Use a consistent set of icons to represent skills and technologies.
